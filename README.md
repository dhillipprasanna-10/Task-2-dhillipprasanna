# Task-2-dhillipprasanna
A feature-rich Python Expense Tracker built with accumulators, lists, dictionaries, file handling, and exception handling. Supports categorized expenses, statistics, monthly reports, CSV data persistence, search, deletion, and detailed summaries through a menu-driven console interface.


A comprehensive and user-friendly Expense Tracker built in Python that helps users record, manage, and analyze their spending habits. This project demonstrates core Python concepts such as accumulators, lists, dictionaries, file handling, exception handling, and modular programming, making it an excellent beginner-to-intermediate level project.

The application provides a menu-driven interface that allows users to add expenses, categorize transactions, generate statistics, search records, create monthly reports, and automatically store data in CSV format for persistence. It also includes robust input validation and detailed summary reports to provide insights into spending patterns.
✨ Features
📌 Add and manage expenses with categories
📊 View statistics (Total, Average, Highest, Lowest)
📅 Generate monthly reports
🔍 Search expenses by category
🗑 Delete individual transactions
♻ Clear all records safely
💾 Automatic CSV data storage and retrieval
📈 Category-wise analysis with percentage contribution
🧮 Implements the Accumulator Pattern for running totals
⚡ Input validation and exception handling
🏗 Modular and well-structured codebase
